# 🔥 Firebase Authentication
Firebase email & password and Google authentication with Jetpack Compose.

## 🧰 Tech stack
- Minimum SDK level 24
- Kotlin based, Coroutines + Flow for asynchronous.
- Jetpack Compose
- MVVM Architecture

## 🗃️ Libraries
- Firebase
- Dagger Hilt
- Lifecycle ViewModel
- Navigation Compose
